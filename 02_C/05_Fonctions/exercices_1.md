# Exercices sur les fonctions

## Exercice 1

Soit le programme suivant :
```c
#include <stdio.h>
int main() {
    int s, a = 1, b = 3 ;
    s = somme(a, b);
    printf("%d + %d = %d\n", a, b, s);
    return 0;
}
```

1. Déclarer et définir la fonction ```somme``` qui effectue la somme des valeurs passées en arguments.
